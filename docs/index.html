What this is

A live daily prediction system for MLB batter props — hit probability and home run probability — built on empirical-Bayes shrinkage, pitcher/park adjustments, and Poisson modeling for rare events. Predictions are logged each morning, graded each night against real outcomes, and compared against live sportsbook odds to surface edge. The numbers below are real — generated from actual games, not a backtest.
