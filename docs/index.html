What this is

Quantified Edge is a live daily prediction system spanning MLB batter props, strikeouts, World Cup match results and anytime goalscorers, Wimbledon (Elo), and F1 qualifying/race outcomes — built on empirical-Bayes shrinkage, opponent/park adjustments, and Poisson-family modeling for rare events. Predictions are logged each morning, graded each night against real outcomes, and compared against live sportsbook odds to surface edge. The numbers below are real — generated from actual games, not a backtest.
